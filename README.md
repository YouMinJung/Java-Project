# Java-Project
Student Information Management Program

<p>Environment : Eclipse in Windows<p>
<p>Language : Java/GUI</p>

<br>

- Java의 GUI를 이용하여 학생 정보 관리 시스템을 구현
- Student_stack이라는 자료구조 생성 (배열 속의 배열-> 학기/강의 분야별 점수 및 학점 관리)
